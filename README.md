# vue + websocket + node + mongodb 构建聊天室

## 安装项目依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 打包
```
npm run build
```

### 规范文件
```
npm run lint
```

### 启动node服务端
```
npm run node
```

### 开启MongoDB 

#### 建库
```
use chat 
```

#### 包含两张表 users 表和messages表 查表
```
db.users.find() 
db.messages.find()
```
